## Usage

**Reccomend to use a venv**

Linux
```
git clone https://github.com/foxos21/ProgrammingQuotes.git
cd ProgrammingQuotes/Quotes
python3 -m venv .venv
status venv/bin/activate
pip install -r requirements.txt
python3 main.py
```
Windows
(not tested)
```
git clone https://github.com/foxos21/ProgrammingQuotes.git
cd ProgrammingQuotes/Quotes
python3 -m venv .venv
.\venv\Scripts\activate
pip install -r requirements.txt
python3 main.py
```
